下载roarm_ws_em0:git clone https://github.com/DUDULRX/roarm_ws_em0.git
	
创建功能包：ros2 pkg create 《功能包名称》 --build-type ament_python --dependencies rclpy std_msgs --license MIT

roarm_ws_em0：
	1、授予串口权限：sudo chmod 666 /dev/ttyUSB0

	2、运行驱动节点：ros2 run roarm_driver roarm_driver
	
		驱动节点位置：/roarm_ws_em0/install/roarm_driver/lib/roarm_driver/roarm_driver
		
		驱动运行是后续节点运行必需的基础
			
	3、运行关节控制面板和rviz2仿真模拟器：ros2 launch roarm_description display.launch.py
	
		节点位置：/roarm_ws_em0/install/roarm_description/share/roarm_description/launch/display.launch.py
			
	4、运行moveit2逆运动学求解：ros2 launch roarm_moveit interact.launch.py 
	
		节点位置：/roarm_ws_em0/install/roarm_moveit/share/roarm_moveit/launch/interact.launch.py
		
	5、键盘控制机械臂
	
		启动机械臂节点控制：ros2 launch moveit_servo demo.launch.py
		
		新终端运行？？？：ros2 run roarm_moveit_cmd setgrippercmd
		
		接受键盘控制命令：ros2 run roarm_moveit_cmd keyboardcontrol

开启临时交换分区避免重构（colcon）时内存不足崩溃：
	创建：	
		sudo fallocate -l 8G /swapfile
		sudo chmod 600 /swapfile
		sudo mkswap /swapfile
		sudo swapon /swapfile
		free -h
	删除：
		sudo swapoff /swapfile
		sudo rm /swapfile

重构工作空间：colcon build --symlink-install --parallel-workers 1
		
配置环境变量：source install/setup.bash
